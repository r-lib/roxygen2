# Access metadata about built-in or available tags

Access metadata about built-in or available tags

## Usage

``` r
tags_list(built_in = TRUE)

tags_metadata()
```

## Arguments

- built_in:

  Logical. Whether to restrict the result to built-in tags (`TRUE`) or
  to extend it to available tags (`FALSE`).
