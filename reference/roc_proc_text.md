# Process roclet on string and capture results

Useful for testing.

## Usage

``` r
roc_proc_text(roclet, input, wd = NULL)
```

## Arguments

- roclet:

  Name of roclet to use for processing.

- input:

  Source string

- wd:

  Working directory
