# Tags for documenting S3

Learn the full details in
[`vignette('rd-S3')`](https://roxygen2.r-lib.org/articles/rd-S3.md).

Key tags:

- `@method ${1:generic} ${2:class}`: If ambiguous, clarify which generic
  an S3 method belongs too. Only needed in very rare cases.

## Usage

``` r
#' @method ${1:generic} ${2:class}
```
